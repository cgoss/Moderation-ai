## GitHub CLI Commands Reference

### View Issues
```bash
# List all issues
gh issue list -R cgoss/Moderation-ai

# List issues by label
gh issue list -R cgoss/Moderation-ai --label phase-1
gh issue list -R cgoss/Moderation-ai --label twitter

# List issues by assignee
gh issue list -R cgoss/Moderation-ai --assignee @me

# List closed issues
gh issue list -R cgoss/Moderation-ai --state closed
```

### Create Issues
```bash
# Create new issue
gh issue create -R cgoss/Moderation-ai \
  --title "Issue Title" \
  --body "Issue description" \
  --label "documentation,phase-1"
```

### Update Issues
```bash
# Update issue status
gh issue close -R cgoss/Moderation-ai 42

# Add label
gh issue edit -R cgoss/Moderation-ai 42 --add-label "in-progress"

# Assign issue
gh issue edit -R cgoss/Moderation-ai 42 --assignee @me
```

### Project Management
```bash
# List projects
gh project list -R cgoss/Moderation-ai

# Add issue to project
gh project item-add 7 --url https://github.com/cgoss/Moderation-ai/issues/42 --owner cgoss
```
